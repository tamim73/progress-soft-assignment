Get Started
  1. Modify the 'Default' connection string in appsettings.json to match your environment
  2. Select the Web project and run 'update-database' command in NuGet Package Manager Console
  3. Run the application by pressing F5
Description
  • Application users have two roles, administrator role and user role.
  • User role
    • List, search and view employees details.
    • Export reports to excel.
  • Administrator role
    • List, search and view employees details.
    • Export reports to excel.
    • Add, edit, delete employees.
    • Disable and enable application users accounts.
    • Change the role of application users.
  • There is an account with username: 'sysadmin' and password: 'p@ssw0rd' that can not be modified.
Features
  • Fully responsive.
  • Server-side pagination, sorting, searching.
  • Jwt role-based authentication.
  • Component-based design with clean architecture.
